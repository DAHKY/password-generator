# Password Generator

A simple and customizable password generator built with Python.  

## Features
- Adjustable password length.
- Include or exclude:
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters
- Generate readable or strong passwords.
- Optionally save generated passwords to a file.
